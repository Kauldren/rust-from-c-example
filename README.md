## Requirements

* yasm (1.1.0+)
* rust (0.8-pre)
* clang (3.4 ... 3.1+ might work)
* grub-mkrescue and xorriso
* binutils for gcc (uses -m32 flag instead of cross compiling)

You will have to build rust from master at https://github.com/mozilla/rust
